Реализовать HTTP-клиент(библиотека) на Go, который способен осуществлять потоковую (условно считаем, что размер передаваемых данных превышает размер доступной для нашего процесса оперативной памяти) передачу данных на HTTP-сервер через POST multipart/form-data запрос.
Размер блока передачи данных можно задавать в конфигурации. По умолчанию 256 байт.
HTTP Клиент должен базироваться на fasthttp.
Исходные данные для передачи можно взять из файла в локальной файловой системе.
После реализации предполагается устная дискуссия на тему практик и подходов, которые кандидат предлагает использовать в подобной задаче если она будет использоваться в продуктивном высоко-нагруженном решении.
Реализация HTTP-сервера может отсутствовать или быть простейшей (чисто для удобства тестирования)
